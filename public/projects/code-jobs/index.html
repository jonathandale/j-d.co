Github repo Web app

Rationale

Once upon a time I was looking for a job and thought it'd be a good idea to consolidate job searches and view a single listing. At the same time I was interested in learning React/Redux, so put this web app together.

Tools

  • React
  • Redux
  • Design

Jobs are pulled from Github API. At the time of development, the StackOverflow jobs xml feed was also parsed & included.
